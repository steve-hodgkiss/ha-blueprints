# ha-blueprints

Collection of Home Assistant Blueprints to assist in automation

## Freeview SD > HD channel switcher for London

Uses a media_player mapped to a TV to detect which channel is currently active.
If there is a better HD channel then automatically change to it.
Blueprint has Freeview channel mapping for London

| SD | HD | Channel |
|---|-----|---------|
| 1 | 101 | BBC One → BBC One HD |
| 2 | 102 | BBC Two → BBC Two HD |
| 3 | 103 | ITV → ITV HD |
| 4 | 104 | Channel 4 → C4 HD |
| 5 | 105 | Channel 5 → 5 HD |
| 9 | 106 | BBC Four → BBC Four HD |
| 201 | 203 | CBBC → CBBC HD |
| 202 | 204 | CBeebies → CBeebies HD |
